Create a customizable slider with your own content blocks.

== Description ==

The Content Slider block allows you to create slides with your own custom content powered by Swiper.js. This block is perfect for creating slideshows, carousels, or galleries with various types of content.

= Features =

* Create custom slides with any WordPress blocks
* Two display modes: Carousel and Fullwidth Slider
* Responsive design that works on all devices
* Customizable slider settings (slides per view, spacing, loop)
* Smooth animations and transitions
* Navigation and pagination controls
* Autoplay option

== Usage ==

1. Add the Content Slider block to your page
2. Choose between Carousel or Fullwidth Slider display mode
3. Add content to individual slides using any WordPress blocks (text, images, buttons, etc.)
4. Customize the slider settings in the block sidebar

== Frequently Asked Questions ==

= How do I add content to my slides? =

The Content Slider block acts as a container for individual slide blocks. Click the "+" button inside each slide to add your desired content blocks.

= Can I customize the appearance of each slide? =

Yes, you can add any WordPress blocks to your slides, including columns, groups, and media blocks, allowing for complete customization of each slide's content and appearance.

= Does the slider work on mobile devices? =

Yes, the Content Slider is fully responsive and will automatically adjust the number of visible slides based on the screen size in carousel mode, or display properly formatted fullwidth slides on mobile devices.
