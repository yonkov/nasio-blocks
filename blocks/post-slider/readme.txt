Display your latest posts in a beautiful responsive slider.

== Description ==

The Post Slider block allows you to showcase your latest posts in an elegant, responsive slider powered by Swiper.js. This block is perfect for highlighting featured content, recent posts, or specific post categories on your website.

= Features =

* Responsive design that works on all devices
* Filter posts by category
* Customizable number of posts to display
* Show/hide featured images, excerpts, author, and date
* Customizable slider settings (slides per view, spacing, loop)
* Smooth animations and transitions
* Navigation and pagination controls

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/post-slider` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the block editor to add the Post Slider block to your page or post

== Frequently Asked Questions ==

= How do I add the Post Slider to my page? =

Simply edit a page or post with the block editor, click the "+" button to add a new block, and search for "Post Slider". Add the block and customize its settings in the sidebar.

= Can I customize how many posts appear in the slider? =

Yes, you can set the number of posts to display, filter by category, and adjust how many slides appear at once on different screen sizes.

= Does the slider work on mobile devices? =

Yes, the Post Slider is fully responsive and will automatically adjust the number of visible slides based on the screen size.
